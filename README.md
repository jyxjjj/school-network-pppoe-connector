# school network pppoe connector

此应用使用Unlicense进行授权，这意味着作者将放弃此应用的版权
原因是此应用过于简单基础
此应用展示了中国江苏南京如何使用PPPOE连接入校园网
使用此应用以替代天翼客户端

### 原因（特别警告）
天翼客户端的驱动程序发现恶意代码
可能导致蓝屏和卡顿等问题
且防止反编译的过程代码疑似病毒行为
可能导致无法卸载等问题

### 使用方式

``` cmd
#请手动新建空账号密码的宽带连接后使用
#修改代码中的账号17300000000为自己的手机号
#双击PPPOE.EXE
#输入6位数动态密码并回车
#系统将弹出RASDIAL命令界面
#此时，网络应该已正常连接
#使用CMD输入
Rasdial.exe 宽带连接 17300000000@njxy 000000
Rasdial.exe 此处是宽带连接的快捷方式名称 此处是手机号@njxy 此处是6位动态密码
#即可实现本程序一致的效果
```
